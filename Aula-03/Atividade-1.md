**Atividade 1 – Plano Orçamentário e Levantamento Técnico**

1. Objetivo

- Implantar uma rede industrial para interligar máquinas de produção, sensores, CLPs e sistemas supervisórios, garantindo:
- Monitoramento em tempo real
- Maior controle produtivo
- Redução de falhas
- Integração futura com ERP corporativo

2. Levantamento Técnico da Infraestrutura

| Item        | Funções / Características                         |
| ----------- | ------------------------------------------------- |
| Comunicação | PROFINET integrada                                |
| Expansão    | Suporte a módulos de I/O                          |
| Programação | Configuração via software de engenharia           |
| Aplicações  | Controle de prensas hidráulicas                   |
|             | Controle de esteiras transportadoras              |
|             | Leitura de sensores de umidade e compactação      |
|             | Controle de processo de secagem                   |
| Finalidade  | Automação de processos industriais de médio porte |
